# Project Name: Ayrshaer CMS

## Description
Ayrshaer CMS is a cross-platform mobile app for content management, enabling users to efficiently manage articles, images, and products with integrated payment, analytics, AI-driven personalization, and advanced features like AI content creation and predictive analytics.

## Primary Goals
- Enable seamless content, media, and product management.
- Integrate Stripe for payment processing.
- Utilize Google Analytics for tracking and analytics.
- Incorporate Gemini AI for content recommendations and personalization.
- Provide a freemium SaaS model targeting individual creators and businesses.
- Introduce AI-driven content creation tools and predictive analytics.
- Enhance security with biometric authentication.
- Support multi-language content management.
- Offer API integrations for third-party services.

----------------------------------------------------

## Tech Stack & Environment

**Frontend:** Vite + React + TypeScript + Tailwind CSS

**Backend:** Supabase (PostgreSQL + Auth)

**Target Platform:** Single-Page Application

**Framework Versions & Config:** Latest stable versions

## Data Model & Supabase Setup

### Database Schema
- Article Management (Title, Content, Tags, Status, Language)
- Media Library (File Name, Type, URL, Metadata)
- Product Catalog (Name, Description, Price, SKU, Inventory)
- User Accounts (Username, Email, Password, Subscription Tier, Biometric Data)

### Row-Level Security
Enabled by default

### Auth Requirements
- Email and password authentication with multi-factor options.
- OAuth integration for social media logins.
- Role-based access control for different user tiers.
- Biometric authentication for enhanced security.

### API Integration
- Third-party API integration support for extending functionality and services.
